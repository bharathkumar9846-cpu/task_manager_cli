# Task Manager CLI

A command-line based task manager built using Python.  
This application allows users to manage tasks efficiently with features like adding, viewing, marking as done, and saving tasks.

---

## 🚀 Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Save tasks to file (JSON)
- Load tasks from file
- Logging support
- Exception handling

---

## 🛠️ Tech Stack
- Python
- JSON (File Storage)
- Logging Module
